Based on the public free API you can find https://documenter.getpostman.com/view/1134062/T1LJjU52#intro you need to build a backend app with the following endpoints:

1. An endpoint that takes a country as a parameter (e.g. Italy, Nigeria, …) and returns:
population
capital city
location
currency
ISO2&3

2. An endpoint that takes a country as a parameter (e.g. Italy, Nigeria, …) and returns the full list of all the states in the country and all the cities in each state.


Purpose of this API is to help you understand how to integrate with other third party APIs. First you need to understand the API, test that the end point you need is available and if not, combine other endpoints to achieve what you want to achieve, the integrate into the API
